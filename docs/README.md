# TabLocker Docs

TabLocker is a cross-browser (Chrome, Edge, Firefox) Manifest V3 extension being built to lock a
tab to a home page and track that tab's browsing history from that point. The project is still in
early scaffolding, and the behavior described here is planned rather than fully implemented yet.

- [Installation](installation.md) — build the extension and load it unpacked for development.
- [Usage](usage.md) — planned features and their current status.
