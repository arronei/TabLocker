# Usage

TabLocker's feature set is still being built out. This page lists the planned features and their
UI entry points; each section will be filled in as the feature ships. See issue #27 for the
originating design.

## Popup menu

Clicking the toolbar icon opens a popup menu with entries for Go Home, History, About, Set Home
Page, and Edit Homepage.

Not yet implemented.

### Go Home

Navigates the current tab to its locked home page.

Not yet implemented.

### Set Home Page

One-click action that captures the current tab's URL as its new home page.

Not yet implemented.

### Edit Homepage

Opens a textbox pre-filled with the current home URL, editable, with a Save button.

Not yet implemented.

### History

Two-tier history access:

- A quick flyout view, reached from the popup, with a search box and a flat list of the current
  tab's tracked history.
- A full history page (`src/history`), reached via a "..." menu entry in the flyout.

Not yet implemented.

### About

A simple view showing version and extension info.

Not yet implemented.
